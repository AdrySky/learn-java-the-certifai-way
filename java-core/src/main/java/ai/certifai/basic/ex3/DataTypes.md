# Types of Data

**What To Do**  
Define these 4 values in the respective format and get it evaluated  
  
- "Nasi Lemak!" as String
- 2 as int
- 100.054 as double
- true as boolean

**Sample Input**  
`
String inputString = "Nasi Lemak!";
eval.evaluatePerLine(inputString);
`
**Sample Output**  
`
Nasi Lemak!
`
**Key Takeaways**

- Different data types in Java: 
    - int
    - double
    - String
    - boolean
